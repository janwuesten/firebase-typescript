# Basic types

## String, Numbers and Booleans

Strings can easily be defined by using `string`, `number` or `boolean` as a type when using the `define` method.

```ts
import { DocumentClass, DocumentClassDefineProps } from "@janwuesten/firebase-typescript"

export class MyDocumentClass extends DocumentClass {
  // highlight-start
  myString!: string
  myNumber!: number
  myBoolean!: boolean
  // highlight-end

  definition({ define, defineCollection, defineHandler }: DocumentClassDefineProps) {
    // highlight-start
    define<typeof this.myString>("myString", "string", "my default value")
    define<typeof this.myNumber>("myNumber", "number", 0)
    define<typeof this.myBoolean>("myBoolean", "boolean", false)
    // highlight-end
  }
}
```

## Date and Timestamps

Dates can be used by defining a prop of the type `Date` and defining the prop with the type `timestamp` when using the `define` method.

```ts
import { DocumentClass, DocumentClassDefineProps } from "@janwuesten/firebase-typescript"

export class MyDocumentClass extends DocumentClass {
  // highlight-start
  myTimestamp!: Date
  // highlight-end

  definition({ define, defineCollection, defineHandler }: DocumentClassDefineProps) {
    // highlight-start
    define<typeof this.myTimestamp>("myTimestamp", "timestamp", new Date())
    // highlight-end
  }
}
```

## Geopoints

Geopoints can be used with the default `GeoPoint` class provided by the Firebase SDK and using the type `geopoint` when using the `define` method.
